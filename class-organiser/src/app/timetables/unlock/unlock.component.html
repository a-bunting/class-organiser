Unlock Data Collection
Make sure that you have done the following before you start collecting data. To miss any one of these checkpoints may mean you need to create a new timetable if you make any changes.

The following items are strongly recommended before unlocking:

  • All the courses students have to choose from should be defined in the 'courses' section.
  • If participants need to be related to a leader, these should also be defined in the 'groups' section.

The following items are NOT required before unlocking and can be changed later:

  • Room or location names can be added at any point.
  • No student data needs to be defined before collecting data.
  • Courses which are checked as 'required' can be added at any time.

The following items are strongly recommended before unlocking:

  • All students must be defined in the 'Student Data'. You do not need data except at least their name.

The following items are NOT required before unlocking and can be changed later:

  • Room or location names can be added at any point.
